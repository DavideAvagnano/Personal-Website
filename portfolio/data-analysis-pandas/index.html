Project

Data Driven Analysis in Pandas

01. Data Import

The project began with importing data from the Kaggle dataset using Python and the Pandas, NumPy, Matplotlib, and Seaborn libraries.

02. Data Cleaning

A crucial phase was the data cleaning process to correct errors or anomalous parameters in the dataset. For example, there was an issue with the 'Rating' column, which contained an anomalous value.

03. Data Analysis and Visualization

Subsequently, an exploratory data analysis was conducted, visualizing and extracting relevant information.

04. Conclusions

After the analysis various considerations have emerged that could be useful for stakeholders interested in developing and launching an application.

The analysis carried out provides interesting insights and leaves room for further exploration. New questions can be explored, and the analysis can be improved by integrating additional data or applying more advanced approaches.

● Sectors with higher and lower competition.

● Analysis of app success based on installations, ratings, and reviews.

● Percentage of free and paid apps.

● Analysis of pricing and adopted business models.
